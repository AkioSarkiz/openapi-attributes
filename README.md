# OpenAPI PHP Attributes

## Features

- [x] Generate info
- [x] Generate schema
- [x] Generate paths
- [x] Generate security schema
- [x] Generate servers


This CLI Tool is able to generate an OpenApi JSON file description according to PHP attributes contained in your files.
